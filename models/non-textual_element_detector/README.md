# AnnoPage Non-Textual Element Detector

This directory contains the trained model for detecting non-textual elements in documents, such as images, tables, graphs, etc. It is YOLO11s model trained on AnnoPage Dataset (see [AnnoPage Dataset: Dataset of Non-Textual Elements in Documents with Fine-Grained Categorization](https://arxiv.org/abs/2503.22526) and [AnnoPage Dataset](https://doi.org/10.5281/zenodo.12788419)).
